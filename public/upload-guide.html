How To Build Game On PC

Create game on computer, export JSON package, upload into app.

Best language

Use JavaScript + HTML + CSS. This is the main supported format because uploaded game runs inside browser frame (WebView).

You can code in VS Code, Cursor, WebStorm, or any editor.

PC Steps

1. Create filegame.html

2. Put your HTML/CSS/JS game codeinside file

3. Create package filemy-game.json

4. Add title/description/htmlContentJSON format

5. Upload on pageUpload Game

JSON Package Template

{
  "title": "My Cool Game",
  "description": "Short game description",
  "htmlContent": ""
}

Important rules

No backend code in packageClient only

Keep size moderateunder 180 KB HTML

Mobile controls recommendedtouch + drag

Use safe JSno external dangerous scripts
